# Campus LMS

 This digital school portal provides a stunning user experience with professional branding, advanced navigation, and full accessibility support.

## Key Features

*   **Vibrant University Branding**: Deeply integrated Zetech Primary Blue and Red color scheme.
*   **Modern Aesthetic**: Built using Material Design 3, featuring stylish rounded curves, "glowing" UI elements, and high-impact layouts.
*   **Dual Theme Support**: Full support for both **Light and Dark modes** with a dedicated manual toggle in the top app bar.
*   **Professional Navigation**: Implemented using Jetpack Navigation components with a persistent bottom navigation bar and intuitive "Back" buttons across all screens.
*   **Interactive Course Portal**: A vertical scrollable list of Technology and Accounting programs with detailed course descriptions in Material Dialogs.
*   **Smart Enrollment**: A streamlined registration form featuring a professional exposed dropdown menu for course selection.
*   **Accessibility for All**: Optimized for screen readers (TalkBack) with full content descriptions and high-contrast visuals.
*   **Smooth Animations**: Sophisticated entrance animations (`fade_in_up`) and button pulse effects for a high-end feel.

## Application Screens

### 1. Login Page
The entry point to the portal, featuring a perfectly circular, enlarged university logo, secure authentication fields with glowing placeholders, and a modern "Login to Portal" action.

### 2. Dashboard Home
A high-impact landing page featuring a large, professionally scaled campus building graphic with a dark gradient overlay, displaying the university motto and mission statement in a floating architectural card.

### 3. Student Registration
A dedicated data entry screen for capturing new student profiles, utilizing an intelligent dropdown menu to simplify course selection and ensure data accuracy.

### 4. Available Programs
A scrollable list of department courses organized into stylish Material cards. Each item is interactive, allowing students to view specific program details upon selection.

## Technical Stack

*   **Language**: Kotlin
*   **Framework**: Android Jetpack (Navigation, Fragments, Material 3)
*   **Layouts**: XML (ConstraintLayout, LinearLayout, CoordinatorLayout)
*   **Animations**: Custom Pulse and Fade-In-Up animations
*   **Build System**: Gradle (KTS)
*   **Compatibility**: Android 12.0+ (API 31+)

## How to Run

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/CampusLms.git
    ```
2.  **Open in Android Studio**: Open the project folder and allow Gradle to sync.
3.  **Build**: Select `Build > Make Project`.
4.  **Run**: Click the `Run` button to deploy the app to your emulator or connected physical device.
